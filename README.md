# Recipe-Web-Application
The recipe application is a web based application for searching and finding recipes online, built using React.js frontend JavaScript library.
The application basically assists you to search recipes online and get results spoken by the device also. This project is good to start with as it helps to understand how react.js works as well as how to create responsive wiebsites.
